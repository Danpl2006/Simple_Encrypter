# Simple Encrypter 🔐

Простое приложение для шифрования и расшифрования текста с использованием авторского шифра Dencrypt.

## Описание

Simple Encrypter — это десктопное приложение с графическим интерфейсом на Kivy, которое позволяет:
- Шифровать текст на русском или английском языке
- Расшифровывать текст обратно
- Выбирать ключ шифрования (0 или 1)
- Копировать результат в буфер обмена
- Переключать язык интерфейса (Русский / English)

## Как работает шифр

- **Русский алфавит** — 33 буквы (с Ё)
- **Английский алфавит** — 26 букв
- **Кодовая таблица**:
  - Для русского: `0-9 A-W` (33 символа)
  - Для английского: `0-9 A-P` (26 символов)
- **Ключ 0** — используется обратный порядок алфавита (Я→А или Z→A)
- **Ключ 1** — используется прямой порядок алфавита (А→Я или A→Z)
